# Three.js

> Three.js 是一个强大的 JavaScript 3D 库，让在网页上创建和显示 3D 图形变得简单。

## 🎯 简介

Three.js 是一个基于 WebGL 的 JavaScript 3D 库，提供了简单易用的 API 来创建和显示 3D 图形。它封装了 WebGL 的复杂性，让开发者能够专注于创意和用户体验。

## ✨ 核心特性

- **🎨 丰富的几何体**: 内置多种 3D 几何体和材质
- **💡 强大的光照系统**: 支持多种光源类型和阴影效果
- **🎬 动画系统**: 内置动画循环和插值功能
- **🔧 可扩展架构**: 插件化设计，易于扩展
- **📱 跨平台兼容**: 支持桌面和移动设备
- **⚡ 高性能渲染**: 基于 WebGL 的高效渲染
